# React Hooks

This project is for teach the pincipal native hooks of React and the principal hooks of react-router-dom

## React Native Hooks

- **useState**
- **useRef**
- **useMemo**
- **useEffect**
- **useContext**

## React-router-dom Hooks

- **useMatch**
- **useLocation**
- **useHistory**
